PS4 Controller
==============
Axis 0: 128     L-Axis Horizontal
Axis 1: 128     L-Axis Vertical
Axis 2: 128     R-Axis Horizontal
Axis 3: 128     R-Axis Vertical
Axis 4: -32768  L2
Axis 5: -32768  R2

Button 0:  CROSS (Green)
Button 1:  CIRCLE (Red)
Button 2:  SQUARE (Orange)
Button 3:  TRIANGLE (Green)
Button 4:  SHARE
Button 5:  Center (Home)
Button 6:  Options
Button 7:  L-Axis
Button 8:  R-Axis
Button 9:  L1
Button 10: R1
Button 11: Pad

Hat 0: D-Pad  (1:N, 2:E, 4:S, 8:W)
