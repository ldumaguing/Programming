flag1
=====
bit 0:   image is a chit (158x158 pixels)
bit 1:   combatant
bit 2:   may fire wire guided missiles
bit 4,3:
   01: -1 APF superscript
   10: +1 APF superscript
bit 5:   range effects DO NOT apply.  Normal Range = Max Range.  (white font)
bit 7,6:
   01: -1 HEF superscript
   10: +1 HEF superscript




CREATE TABLE img (
   id   INTEGER PRIMARY KEY,
   name TEXT NOT NULL,
   flag1 NUMBER,
   apf_val NUMBER,
   apf_rng TEXT,
   hef_val NUMBER,
   hef_rng TEXT
);
